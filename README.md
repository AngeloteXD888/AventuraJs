# Aventura en el Reino de JS

## Descripción

**Aventura en el Reino de JS** es un videojuego RPG basado en navegador donde el jugador lucha contra oleadas de enemigos en una arena mística para alcanzar la máxima puntuación. La clave del éxito radica en la selección estratégica de objetos en el mercado al inicio de cada aventura.

## Características Principales

- **Sistema de Combate por Turnos**: Enfrenta a enemigos regulares y jefes con multiplicadores especiales
- **Mercado Interactivo**: Compra armas, armaduras y consumibles con descuentos aleatorios
- **Sistema de Estadísticas**: Mejora tu ataque, defensa y vida con el equipamiento adecuado
- **Sistema de Ranking**: Alcanza el rango de Veterano superando el umbral de puntuación
- **Interfaz Visual**: Experiencia completamente visual sin uso de consola

## Tecnologías Utilizadas

- JavaScript ES6+ (Módulos)
- HTML5
- CSS3 (Grid, Flexbox, Animaciones)
- Programación Orientada a Objetos
- JSDoc para documentación

## Cómo Jugar

1. **Inicio**: Visualiza tus estadísticas iniciales
2. **Mercado**: Compra armas, armaduras y consumibles con descuentos aleatorios
3. **Preparación**: Revisa tu inventario y estadísticas actualizadas
4. **Combate**: Enfrenta a enemigos en combates por turnos
5. **Final**: Descubre tu puntuación final y tu rango

## Instalación y Uso

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. ¡Comienza tu aventura!
